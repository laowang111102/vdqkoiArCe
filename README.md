# 新房数据可视化及预测系统 python713

## 项目概述

本项目是一款基于Python和Flask框架，结合机器学习技术的新房数据可视化及预测系统。系统通过对链家网新房数据的爬取、处理和智能分析，为用户提供直观的数据展示和房价预测功能。

## 技术栈

- **数据爬取**：使用Requests库突破反爬机制，实现数据的爬取。
- **数据存储**：采用MySQL进行智能存储和管理数据。
- **Web交互**：基于Flask框架搭建的Web交互界面，提供用户操作接口。
- **机器学习**：应用Scikit-learn库进行数据处理和模型训练。

## 功能模块

- **数据爬取模块**：18维全字段爬取，包括房名、封面图、市区、户型、装修、预售证、单价、总价区间、开盘交房时间等信息。
- **数据存储模块**：高效存储爬取数据，便于后续处理和分析。
- **数据可视化模块**：将数据以图表形式直观展示，便于用户理解。
- **房价预测模块**：利用机器学习算法进行房价预测。

## 系统角色

- **用户**：通过Web界面进行数据的查询、可视化和预测。
- **系统**：提供数据爬取、存储、处理和预测的服务。

## 运行环境

- **后端**：Python 3.x
- **Web框架**：Flask
- **数据库**：MySQL
- **依赖库**：Requests, Scikit-learn等

## 部署步骤

1. 确保所有依赖库已安装。
2. 配置数据库MySQL环境。
3. 导入初始数据至数据库。
4. 运行Flask应用。

- **智能预测**：基于机器学习算法，提供精准的房价预测。
